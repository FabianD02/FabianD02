Arduino Car Project

A project that put coding into a selfdriving car

This is a page talks about how my Arduino Car Project was possibe. The very first and basic step was being able to correctly put wires where they needed to be on the arduino and the mother board. Next we needed to implament a premade code into the car so that it was able to move forward with manual comand. The next step was to modify the code so that the car would drive forward, left, right, and backwards. The Final step was to put a sensor on the car and modify the code once again to use the sensor and detect objects if they are to close so the car is now able to drive on its own and avoid crashing.
